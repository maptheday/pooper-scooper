# First Day Script

Use this when onboarding a new employee or contractor to the route and daily yard service process.

## What to cover
- Where to meet and how to load the truck.
- Route order and how to read the service map.
- How to approach a yard, find the pet waste, and clean it efficiently.
- What to do if a gate is locked, a dog is loose, or a customer is home.
- How to report missed yards, extra work, or safety issues.

## Suggested flow
1. Welcome them and set expectations for the day.
2. Show the tools: scoopers, bags, gloves, disinfectant, and scrub brushes.
3. Walk through one or two yards together.
4. Explain the standard visit time and quality check.
5. Review communication rules for customers and the office.

## Safety notes
- Always keep a safe distance from unknown dogs.
- Never open a gate without permission.
- If a dog is aggressive, leave and report it immediately.
- Wear gloves and change them between yards if needed.

## Good opening
"Today we’re going to cover how to handle recurring service yards, what to do if a gate is locked, and how to leave every customer with a clean, odor-free yard."
